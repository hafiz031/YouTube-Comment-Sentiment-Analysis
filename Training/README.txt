Reference book for this training code: Python Machine Learning - Sebastian Raschka.

The notebook contains the implementation of a logistic regression model which can classify comments as either "Positive" or as "Negative".

The dataset was collected from: http://ai.stanford.edu/~amaas/data/sentiment/

First download the dataset and extract it and copy the folder "aclImdb" in your "Training" directory and follow instructions from the notebook.
